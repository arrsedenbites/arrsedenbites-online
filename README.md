# 🍃 ARR's Eden Bites - Street Food Cart Website

A modern, responsive single-page website for **ARR's Eden Bites**, a street food cart serving delicious waffles, egg kulfi, French fries, chicken items, and fresh juices in **Devarapalle, Andhra Pradesh**.

🌐 **Live Website:** https://arrsedenbites-online.onrender.com

---

## 🚀 Features

* 🏠 **Hero Section** – Eye-catching landing page with call-to-action buttons
* 📖 **About Section** – Story behind ARR's Eden Bites
* 🍽️ **Menu Section** – Filterable food menu with categories and item details
* 💳 **UPI Payment** – Direct payment via BharatPe QR Code
* ⭐ **Google Reviews** – One-click link to leave a review
* 📍 **Google Maps Integration** – Easy navigation with operating hours
* 📞 **Contact Section** – Clickable phone number, email, WhatsApp, and Instagram
* 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop
* 🎨 **Modern UI** – Food-inspired black, red, orange, and yellow theme
* ⚡ **Fast & Lightweight** – Built using pure HTML, CSS, and JavaScript

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/arrsedenbites/arrsedenbites-online.git
```

### 2. Navigate to the Project Folder

```bash
cd arrsedenbites-online
```

### 3. Run the Website

Simply open **index.html** in your preferred web browser.

Or, if you're using **Visual Studio Code**, install the **Live Server** extension and click **Go Live**.

> No installation, build tools, or dependencies are required.
